Información

Esta aplicación web está basado en un backend en Node.js, que almacena información en formato Json en una base de datos en MongoDB. A esta base de datos se accede mediante peticiones API Restfull, las cuales son visualizadas en el frontend hecho en AngularJS.

Toda la información mostrada en Jurado Alvarez es dinámica y está sujeta a lo almacenado en la base de datos. Todo lo que sea añadido o eliminado en el controllador será actualizado en la página principal.

API

GET al conjunto completo:

URL/api/jurado/dex

GET a un tipo concreto:

URL/api/jurado/dex/tipo
